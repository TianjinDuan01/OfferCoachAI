# OfferCoach AI

> **"They let you practice. We make you better."**

An AI-powered interview coaching platform that generates company-specific questions, evaluates answers in real time, detects STAR structure gaps, and tracks skill growth across sessions.

---

## Demo

**Live:** [\[GitHub Pages link here\]](https://github.com/TianjinDuan01/OfferCoachAI)

**Or run locally:** download `index.html` and open in any browser — no installation required.

---

## What it does

| Feature | Description |
|---------|-------------|
| **CompanyDNA** | Detects T1 / T2 / T3 interview patterns from company name + JD using Claude API |
| **Question generation** | Generates 3 role-specific behavioral questions tailored to the JD |
| **ResumeMatch** Pro | Maps your resume to each question and builds a personalized STAR scaffold |
| **4-dimension scoring** | Evaluates every answer across Logic, Specificity, Role Relevance, and Clarity |
| **STAR detection** | Identifies which components (Situation / Task / Action / Result) are missing or weak |
| **Radar growth tracking** | Saves scores across sessions in localStorage and visualizes improvement over time |

## Quickstart

### Method 1
1. Open `index_presentation.html` in a text editor
2. Replace `YOUR_API_KEY_HERE` on line 1 with your Anthropic API key
3. Double-click to open in browser

### Method 2
1. Rename `index_submission.html` → `index.html`
2. Open in browser — enter your Anthropic API key when prompted
3. Or click **"Skip — use demo mode"** to explore without a key

> Get a free API key at [console.anthropic.com](https://console.anthropic.com)

## Demo flow (2–3 minutes)

1. **Landing page** → click "Start for free"
2. **Session Setup** → click "Load Demo" (blue banner at top)
   - Pro activates automatically
   - Elina's resume pre-loaded
   - Amazon BA I JD pre-filled
   - CompanyDNA shows T1 · High Match · 94%
3. Click **"Analyze JD & Generate Questions"** → jumps to Coaching Chat instantly
4. **Q1 answer is pre-filled** → click Submit → Claude API evaluates in real time
5. Scores, STAR detection, and coaching suggestion appear on the right dashboard
6. ResumeMatch scaffold (green panel) shows above each question
7. Click **"Next Question"** → Q2 pre-filled → repeat
8. After Q3 → click **"View Radar Report"** → radar chart unlocks with real session data
